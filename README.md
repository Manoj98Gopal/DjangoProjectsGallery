### How to create a virtual environment for specific project

    - pipenv install django==5.2

## How to activate virtual environment of specific project

    - pipenv shell

## How to deactivate virtual environment of specific project

    - exit or Ctrl + D

## How to create django project

    - django-admin startproject (project_name) : it will create two folders with project name
    - django-admin startproject (project_name) . : it will create one folder with project name

## How to run django project 

    - python manage.py runserver


## How to create django app

    - python manage.py startapp (app_name)
    - After creating an app, add app_name in the installed app list from setting.py file
